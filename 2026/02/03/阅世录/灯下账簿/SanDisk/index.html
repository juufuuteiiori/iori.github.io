SanDisk（闪迪） 是一家全球领先的 闪存存储解决方案提供商，其核心业务围绕 NAND Flash 存储技术与产品。商业收入来源企业、个人 SSD。撞到了 AI 的风口，市场需求很足。

NAND 设计 → 联合制造 → SSD/嵌入式模块开发 → OEM/渠道销售 → 企业 & 数据中心解决方案部署
NAND 闪存 是一种 非易失性（non-volatile）存储器，也就是说它可以在 断电后仍然保存数据。
未来的发展方向是堆层，提高密度。和芯片有点像。

HBM（High Bandwidth Memory，高带宽内存）是一种 基于 DRAM 的 3D 堆叠内存，通过 TSV（硅通孔）+ 超宽接口，为 GPU/AI 加速器提供极高带宽、极低延迟的内存。
HBF（High Bandwidth Flash，高带宽闪存）是一种 基于 NAND Flash 的新型存储形态，目标是：用 远高于 SSD 的带宽 + 远高于 DRAM 的容量 + 远低于 HBM 的成本。在 AI 系统中 “补位”内存层级

│
│  HBM        ← 热数据 / 实时计算
│
│  HBF        ← 大模型参数 / KV Cache / 扩展内存
│
│  SSD        ← 数据集 / checkpoint
│
│  HDD / 云
```SRAM（Static Random Access Memory，静态随机存取存储器）是一种 易失性内存，只要通电，数据就能一直保持，不需要刷新。计算机架构课程中讲到的内存分级体系的最高级，直接和 CPU 打交道的 L1 L2 L3
DRAM（Dynamic Random Access Memory，动态随机存取存储器）也是 易失性内存，但必须 不断刷新 才能保存数据。

AI 发展的太快，内存不够用了，于是从 DRAM 继续往下找，找到了 NAND 闪存。于是这个行业的老大闪迪起飞了。本质上还是 AI 叙事？驱动力是数据中心，AI 终端的电脑手机。
