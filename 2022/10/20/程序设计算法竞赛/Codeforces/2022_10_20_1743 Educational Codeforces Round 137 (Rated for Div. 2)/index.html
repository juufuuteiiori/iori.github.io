A.Password （0:04）

题意

从 $10-n$ 个数字中选 $2$ 个数字，这 $2$ 个数字各用两次，组成一个四位数（允许前导 $0$）。

求方案数。

分析

对于任意两个数字 $a$，$b$，能组成的方案为 $aabb$，$abab$，$abba$，$baab$，$baba$，$bbaa$ 共六种方案

方案总数 $ans = C^2_{10-n} \times 6$

Code

void solve()
{
    int n;
    std::cin >> n;
    for(int i = 1; i <= n; i++)
    {
        int a;
        std::cin >> a;
    }
    int ans = 3 * (10 - n) * (9 - n);
    std::cout << ans << '\n';
}
```### B. Permutation Value （0:18）

#### 题意 (重复2)

对于长度为 $n$ 的一个排列，定义 $Value$ 为子段是排列的个数

构造一个排列，让它的 $Value$ 最小

#### 分析 (重复2)

不难发现，$Value$ 最小为 $2$，即 $[1]$ 和 $[1,2,3,\cdots,n]$。

要满足条件，最简单的构造方式为 $[1,n,2,3,\cdots,n-1]$

#### Code (重复2)

```c++
void solve()
{
    int n;
    std::cin >> n;
    std::cout << 1 << ' ' << n << ' ';
    for(int i = 2; i < n; i++)
        std::cout << i << (i == n - 1 ? '\n' : ' ');
}
```### C. Save the Magazines （0:27）

#### 题意 (重复3)

长度为 $n$ 的数组，权值为 $a_i$，权重为 $b_i(b_i = 0,1)$

循环 $i = 2 \to n$，当 $b[i - 1] = 0$ 且 $b[i] = 1$ 时，可以让 $b[i - 1] = 1$ 且 $b[i] = 0$

求 $\max(\sum_{i = 1 \to n} a_ib_i)$

#### 分析 (重复3)

对于 $01$ 的情况，当 $a_0 > a_1$ 时，进行操作
对于 $01111$ 的情况，当存在一个 $j$ 满足 $a_0 > a_{1j}$ 时，进行操作

#### Code (重复3)

```c++
void solve()
{
    int n;
    std::cin >> n;
    std::string s;
    std::cin >> s;
    std::vector<int> a(n);
    for(int i = 0; i < n; i++)
        std::cin >> a[i];
    int ans = 0;
    int L = 0;
    while(L < n)
    {
        if(s[L] == '0')
        {
            int R = L + 1;
            while(R < n)
            {
                if(s[R] == '0') break;
                if(a[R] < a[L]) break;
                R++;
            }
            if(s[R] == '1')
            {
                s[L] = '1';
                s[R] = '0';
            }
            L = R;
        }
        else L++;
    }
    for(int i = 0; i < n; i++)
        if(s[i] == '1') ans += a[i];
    std::cout << ans << '\n';
}
```### D. Problem with Random Tests （1:17）

#### 题意 (重复4)

给一个纯随机的 $01$ 字符串，从中选择两个子串，子串的价值 $Value$ 为将其视为二进制的值

求 $\max(Value1 | Value2)$

#### 分析 (重复4)

毫无疑问，一定会取整个字符串，因为他一定是所有串中最大的，且最高位为 $1$
考虑寻找一个字符串能够尽可能的增大 $Value$，也就是从高位到低位，依次考虑这个位置的 $0$ 能否通过或运算让它变成 $1$

考虑到该字符串为纯随机，每增长一位，备选的字符串数量平均减少一半。

#### Code (重复4)

```c++
void solve()
{
    int n;
    std::cin >> n;
    std::string s;
    std::cin >> s;

    bool flag = false;
    int pos = -1;
    for(int i = 0; i < n; i++)
    {
        if(!flag && s[i] == '1') flag = true;
        if(flag && s[i] == '0')
        {
            pos = i;
            break;
        }
    } // 去掉前导 0 后，寻找第一个 0
    if(pos == -1)
    {
        flag = false;
        for(int i = 0; i < n; i++)
        {
            if(s[i] == '1') flag = true;
            if(flag == true)
                std::cout << s[i];
        }
        if(!flag) std::cout << '0';
        return;
    }
    std::vector<int> a;
    for(int i = 0; i < pos; i++)
        a.push_back(i);// 备选项
    for(int i = 0; i < n - pos; i++)
    {
        if(s[pos + i] == '1') continue;// 这一位是 1
        bool flag = false;
        for(int x : a)
            if(s[x + i] == '1') flag = true;
        if(!flag) continue;// 都无法满足使这一位变成 1
        std::vector<int> b;
        for(int x : a)
            if(s[x + i] == '1') b.push_back(x);// 能满足的保留下来
        a = b;
    }
    int K = a[0];// 随便选一个都行
    flag = false;
    for(int i = 0; i < pos; i++)
    {
        if(s[i] == '1') flag = true;
        if(flag) std::cout << s[i];
    }
    for(int i = 0; i < n - pos; i++)
        std::cout << ((s[pos + i] == '1' || s[K + i] == '1') == 1 ? '1' : '0');
}
```### E. FTL （1:52）

#### 题意 (重复5)

两种武器，各有一个伤害 $p$ 与冷却时间 $t$。
一个敌人，有血量 $h$ 与护甲 $s$。

单独使用武器，能造成伤害 $p - s$。
同时使用两个武器，能造成伤害 $p_1 + p_2 - s$

求击败敌人的最短时间。

#### 分析 (重复5)

预处理计算 $F[x]$ 为造成 $x$ 伤害的最短时间
于是问题转化为无穷背包问题。

对于 $F[x]$ 的计算，枚举两种武器各使用多少次，并且最后一次为同时使用。
花费时间为 `std::max(t1 * i, t2 * j)`
造成伤害为 `(i - 1) * (p1 - s) + (j - 1) * (p2 - s) + (p1 + p2 - s)`，某一个武器使用次数为 $0$ 单独考虑。

#### Code (重复5)
```c++
void solve()
{
    ll p1,p2,h,s;
    ll t1,t2;
    std::cin >> p1 >> t1;
    std::cin >> p2 >> t2;
    std::cin >> h >> s;
    for(ll i = 1; i <= h; i++)
    {
        f[i] = 1e18;
        F[i] = 1e18;
    }
    for(ll i = 0; i <= h; i++)
        for(ll j = 0; j <= h; j++)
        {
            ll P = 0;
            if(i > 0 && j > 0)
                P = (i - 1) * (p1 - s) + (j - 1) * (p2 - s) + (p1 + p2 - s);
            else if(i > 0)
                P = i * (p1 - s);
            else if(j > 0)
                P = j * (p2 - s);
            ll T = std::max(t1 * i, t2 * j);
            if(P > h) P = h;
            F[P] = std::min(F[P],T);
        }
    for(ll i = 0; i < h; i++)
    {
        for(ll j = 1; j <= h; j++)
        {
            ll k = std::min(h,i + j);
            f[k] = std::min(f[k], f[i] + F[j]);
        }
    }
    std::cout << f[h] << '\n';
}
