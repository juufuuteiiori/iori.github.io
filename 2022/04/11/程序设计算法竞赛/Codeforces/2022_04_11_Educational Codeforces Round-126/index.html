A. Array Balancing

题意

给两个数组 $a,b$ ，可以进行若干次交换 $a_i$ 和 $b_i$

令 $\sum_{i=2}^n (|a_i - a_{i-1}|+|b_i-b_{i-1}|)$ 最小

思路

不难发现，每对值只与前面的数字有关系

那么只需要考虑当前是否交换

```## C. Water the Trees

### 题意 (重复3)

给一个数组，通过轮流进行操作 $1$ 和操作 $2$ （开始是操作 $1$），让所有数相同

- 操作 $1$：选择一个数字，给它 $+1$；或者跳过
- 操作 $2$：选择一个数字，给它 $+2$；或者跳过

求最少操作多少次

### 思路 (重复3)

> 比赛的时候犯蠢了，认为对最大值进行操作无意义。。。
>
> 比赛结束后去问了万能的群友，三分钟改完代码AC。。。

考虑到最终所有数字相同，不妨设最终的值 $finalValue=Max$ 或 $finalValue = Max+1$

定义 $c1$ 为操作 $1$ 的次数，定义 $c2$ 为操作 $2$ 的次数

遍历一遍数组，对于每个数字， 为了让它变成 $finalValue$ ，用贪心的思想去理解，容易发现能进行操作 $2$ 就进行操作 $2$

当 $c1 \ge c2$ 时，我们没法操作了， $ans = 2\cdot c1 -1$

当 $c1 < c2$ 时，$ans = 2\cdot c2$，不一定为最优解。考虑将 $c2$ 变小， $c1$ 变大

对于 $c2$ 减少 $x$，则 $c1$ 增加 $2\cdot x$ 。不难发现 $0\le c1 - c2 \le 2$ 时，$ans = c1 + c2$ 最优

这时回过来思考一下，为什么对最大值进行操作存在意义？

这么做是不是相当于令 $c2 = c2 + c1$ ，$c1 = n - c1$。此时我们可以将 $c1 > c2$ 的情况转换成 $c1 < c2$ 的情况，哪怕添加的总值会变大，但我们可以通过采取更优的 $+2$ 来减少操作次数。可以结合例子 $a[]=\{1,1,1,1,1,2\}$ 去理解为什么要考虑增加最大值

```c++
#include<bits/stdc++.h>
#define LL long long
LL cal(LL Max,std::vector<int> &a) {
    LL c1 = 0, c2 = 0, ans = 0;

    for(int x : a) {
        c1 += (Max - x) % 2;
        c2 += (Max - x) / 2;
    }
    if(c1 > c2) ans = 2 * c1 - 1;
    else {
        LL sum = c1 + 2 * c2;
        c2 = sum / 3;
        c1 = sum - c2 * 2;
        ans = c1 + c2;
    }
    return ans;
}
void solve() {
    int n;
    std::cin >> n;
    std::vector<int> a(n);
    for(int i = 0; i < n; i++) std::cin >> a[i];
    int Max = *std::max_element(a.begin(),a.end());
    std::cout << std::min(cal(Max, a),cal(Max + 1, a)) << '\n';
}
int main(){
}
```## D. Progressions Covering

### 题意 (重复4)

给定数组 $b[n]$，对于一个初值为 $0$ 的数组 $a[n]$，进行操作使任意 $a_i\ge b_i$ 成立：

- 选择起点 $i$ ，对于 $a[i]，\cdots，a[i+k-1]$ 分别 $+1,+2,\cdots，+k$。如果 $i+k>n$ ，对于越界下标不考虑

求最少次数

### 思路 (重复4)

不难发现，第 $n$ 个点的最优情况就是每次 $+k$ ，此时对 $a[n-k+1],\cdots,a[n-1]$ 也会产生贡献，能确定的是，这么操作一定不会更劣。（可以自己去想想，也是贪心的思路

那么我们考虑从后往前依次处理

定义 $sum$ 是目前对 $a[x]$ 已经有的贡献，$Num$ 是目前有多少次操作覆盖了 $x$，$cnt$ 是对当前点还需要做多少次操作，$b[x]$ 表示有多少个操作在 $x$ 出结束

简单模拟一下就行

```c++
#include<bits/stdc++.h>
#define LL long long
void solve() {
    int n,k;
    std::cin >> n >> k;
    std::vector<LL> a(n),b(n);
    for(int i = 0; i < n; i++)
        std::cin >> a[i];

    LL ans = 0,sum = 0, cnt = 0, Num = 0, delta;
    for(int i = n - 1; i >= 0; i--) {
        sum -= Num;
        cnt = 0,
        delta = std::min(k,i + 1);
        if(a[i] > sum) {
            cnt = (a[i] - sum + delta - 1) / delta;
            if(i > k) b[i - k] += cnt;
            sum += 1ll * delta * cnt;
        }
        Num = Num - b[i] + cnt;
        ans += cnt;
    }

    std::cout << ans << '\n';
}
int main() {
}
