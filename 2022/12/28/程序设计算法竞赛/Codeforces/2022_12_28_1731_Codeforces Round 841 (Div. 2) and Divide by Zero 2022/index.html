A. Joey Takes Money （0:02）【496/500】

题意

给一个数组 $a$，可以进行任意次操作，每次操作选择两个下标 $i,j$，找到两个数 $x,y$，满足 $x\times y = a_i \times a_j$，用 $x$ 替换 $a_i$，用 $y$ 替换 $a_j$。
使 $\sum a_i$ 最大

分析

考虑两个数字 $a,b$，在 $a\times b$ 不变的条件下，要使 $a+b$ 最大，应该让 $a = 1$ 或 $b = 1$
于是推到 $n$ 个数字，就是让其中 $n - 1$ 个数字变为 $1$

void solve()
{
    int n;
    std::cin >> n;
    i64 res = 1;
    for(int i = 0; i < n; i++)
    {
        i64 x;
        std::cin >> x;
        res = res * x;
    }
    res = res + n - 1;
    std::cout << res * 2022 << '\n';
}
```### B. Kill Demodogs （0：14）【944/1000】

#### 题意 (重复2)

给一张 $n\times n$ 的地图，起点为 $(1,1)$，终点为 $(n,n)$，每次移动可以向右或者向下。
坐标为 $(i,j)$ 的点贡献为 $i\times j$，总贡献为路径上所有点的贡献之和，求最大值。

#### 分析 (重复2)

不难发现，尽可能多得经过对角线能使贡献最大，于是考虑右下右下的方式
贡献为 $1 \times 1+ 1 \times 2 + 2 \times 2 + \cdots + n \times n$

化简一下 $2 \times \frac{n \times (2n - 1) \times (n - 1)}{6} + n ^ 2 + \frac{n \times (n - 1)}{2}$

```c++
void solve()
{
    i64 n;
    std::cin >> n;
    i64 res1 = n * (2 * n - 1) % mod * (n - 1) % mod * ksm(3,mod - 2) % mod;
    i64 res2 = n * (n - 1) / 2;
    i64 res3 = n * n;
    i64 res = (res1 % mod + res2 % mod + res3 % mod) * 2022 % mod;
    std::cout << res << ' '  << '\n';
}
```### C. Even Subarrays （0:37）【1178/1500】

#### 题意 (重复3)

给一个数组 $a$，寻找有多少对 $(i,j)(1\le i \le j \le n)$ 满足 $a_i$ 到 $a_j$ 的异或和的因子个数为偶数个。

#### 分析 (重复3)

不难发现，因数个数为奇数的数都是完全平方数
结合数据范围，考虑用总数减去不合法的情况
用数组 $cnt[x]$ 记录前缀异或和中 $x$ 的出现次数，对于当前的前缀异或和 $sum$，枚举完全平方数 $j \times j$，去掉不合法的情况 $cnt[sum ⊕ (j \times j)]$

```c++
void solve()
{
    int n;
    std::cin >> n;
    for(int i = 0; i < n; i++)
        std::cin >> a[i];
    memset(cnt,0,sizeof(cnt));
    cnt[0] = 1;

    i64 res = 0;
    int sum = 0;

    for(int i = 0; i < n; i++)
    {
        res += i + 1;
        sum ^= a[i];
        for(int j = 0; j * j < N; j++)
        {
            int op = sum ^ (j * j);
            res -= cnt[op];
        }
        cnt[sum]++;
    }

    std::cout << res << '\n';
}
```### D. Valiant's New Map （0:49）【1206/1500】

#### 题意 (重复4)

给一个 $n\times m$ 的矩阵 $A$，找到一个最大的 $l$ ，满足能够从矩阵中选出一块 $l \times l$ 的子矩阵，子矩阵中的所有数字都不小于 $l$

#### 分析 (重复4)

答案有一个很显然的单调性，考虑二分
记当前判断的值为 $x$，新开一个矩阵 $B$，将 $A[i][j] < l$ 的标记 $B[i][j] = 1$，用矩阵 $C$ 记录二维前缀和，枚举一遍所有点，判断是否存在一个二维区间满足区间和为 $0$

```c++
void solve()
{
    int n,m;
    std::cin >> n >> m;
    std::vector map(n,std::vector<int> (m));
    for(int i = 0; i < n; i++)
        for(int j = 0; j < m; j++)
            std::cin >> map[i][j];
    auto check = [&](int x)
    {
        std::vector f(n + 1,std::vector<int> (m + 1));
        for(int i = 0; i < n; i++)
            for(int j = 0; j < m; j++)
                if(map[i][j] < x) f[i + 1][j + 1] = 1;
        for(int i = 1; i <= n; i++)
            for(int j = 1; j <= m; j++)
                f[i][j] = f[i][j] + f[i - 1][j] + f[i][j - 1] - f[i - 1][j - 1];
        for(int i = x; i <= n; i++)
            for(int j = x; j <= m; j++)
            {
                if(f[i][j] + f[i - x][j - x] - f[i - x][j] - f[i][j - x] == 0)
                    return true;
            }
        return false;
    };

    int l = 1, r = std::min(n,m);
    while(l < r)
    {
        int mid = (l + r + 1) >> 1;
        if(check(mid)) l = mid;
        else r = mid - 1;
    }
    std::cout << l << '\n';
}
```### E. Graph Cost

#### 题意 (重复5)

给一张图，有 $n$ 个点，标记为 $1$ 到 $n$，$0$ 条边，要求给图恰好加 $m$ 条边
每次操作时选择一个 $k(k \ge 2)$，之后从图中选出恰好 $k - 1$ 对 $(i,j)$，要求 $gcd(i,j) = k$，其中 $(i,j)$ 不能重复被选择，花费代价为 $k$ 给图加上这 $k - 1$ 条边，求最小代价。

#### 分析 (重复5)

不难发现，每次选择的 $k$ 尽可能大能使代价尽可能小。
于是 $k = n \to 2$ 枚举，问题转换为 $1\to n$中有多少对 $(i,j)$ 满足 $gcd (i,j) = k$
首先算出有多少对 $(i,j)$ 有共同的因数 $k$，设对数为 $sum = (n / k) \times (n / k - 1) / 2$，即从 $k,2k,3k,4k,\cdots,(n / k) * k$中选择两个数的方案
之后去除掉有共同因数 $2k,3k,4k$ 的有序对

```c++
void solve()
{
    i64 n,m,res = 0;
    std::cin >> n >> m;
    std::vector<i64> t(n + 1);
    for(int i = n; i >= 2; i--)
    {
        i64 sum = (n / i) * (n / i - 1) / 2;
        for(int j = 2 * i; j <= n; j += i)
            sum -= t[j];
        t[i] = sum;
    }
    for(i64 i = n; i >= 2; i--)
    {
        i64 k = std::min(t[i] / (i - 1),m / (i - 1));
        res += k * i;
        m -= k * (i - 1);
    }
    std::cout << (m == 0 ? res : -1)<< '\n';
}
