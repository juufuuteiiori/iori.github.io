```### B. Jumbo Extra Cheese 2 （0:14）【956/1000】

#### 题意 (重复2)

给若干个矩形，将这些矩形拼成一个多边形，要求每个矩形都有一条边在 $x$ 轴上。
求这个多边形边长的最小值

#### 分析 (重复2)

假定我们现在已经决定好第 $i$ 个矩形在 $x$ 轴上的那一条边长度为 $a_i$，平行于 $y$ 轴的那一条边长度为 $h_i$。
令 $H=\max(h_i)$
当我们把矩形按高度从小到大依次排列，最终的周长为 $2\times(H + \sum a_i)$ ，找不到一种更优的排列让周长更小。

接着考虑每一个矩形应该如何放置
令 $H$ 为假定的最高值，满足 $a_i\le H,b_i \le H(1\le i \le n)$
对于任意的矩形，假定它的边长 $a \le b \le H$ 时，令长度为 $b$ 的那条边为高，长度为 $a$ 的那条边为底，对于整体周长的增加值 $\Delta =2\times a$ 一定不会有更小的情况。

此时寻找 $H$ 的具体值应该是多少。不难发现 $H$ 就是边长中的最大值。

```c++
void solve()
{
    int n;
    std::cin >> n;
    std::vector<std::pair<int,int>> s(n);
    for(int i = 0; i < n; i++)
    {
        std::cin >> s[i].first >> s[i].second;
        if(s[i].first < s[i].second)
            std::swap(s[i].first,s[i].second);
    }
    std::sort(s.begin(),s.end());
    long long sum = 0;
    for(auto x : s)
        sum = sum + x.second;
    sum = sum + s[n - 1].first;
    sum = sum * 2;
    std::cout << sum << '\n';
}
```### C. Bricks and Bags （0:54）【1141/1500，-2】

#### 题意 (重复3)

有 $n$ 个数字，$3$ 个盒子。
玩家 $A$ 把数字任意放入每个盒子，使得每个盒子至少有一个数字。
玩家 $B$ 从每个盒子中取出一个数字 $w_i$
记得分 $score = |w_1-w_2| + |w_2-w_3|$

已知玩家 $B$ 的策略是最小化 $score$
求 $score$ 的最大值

#### 分析 (重复3)

考虑 $2$ 号盒子中只有一个数字 $a_i$ 的情况
此时玩家 $A$ 需要把剩余 $n-1$ 个数字分成两堆。
无论玩家 $A$ 如何分这两堆数字，玩家 $B$ 总能选择一个数字 $a_j$，满足对于任意的 $k\neq i,k\neq j$ 有 $|a_i - a_j| \le |a_i - a_k|$。
于是玩家 $A$ 的目标是最大化另一个数字产生的价值。也就是让玩家 $B$ 在这一个盒子中没有其他选择除了选择这一个产生最大价值的数字，即盒子中只有一个数字。不难发现，这个数字要么是 $n$ 个数字中的最小值，要么是最大值（此时不考虑边界冲突的问题，即假设最小值和最大值都是可以使用的状态）。

在这种情况下，不难发现被选择的这个数字 $a_i$ 本身就是最大值或最小值时 $score$ 最大。因为有一个完整的 （最大值 - 最小值） + （最大值-次大值 或 次小值-最小值），其余情况都是 （最大值 - 中间值 或 中间值 - 最小值） + |中间值 - 相邻值|

再考虑 $2$ 号盒子中有 $k$ 数字的情况，这 $k$ 个数字要么是最小的 $k$ 个数字，要么是最大的 $k$ 个数字。
此时玩家 $B$ 会选择 最小的数字中最大的那个 或 最大的数字中最小的那个。为了让这个数字与 $1$ 号盒子还有 $3$ 号盒子中的数字差值尽可能的小。
按照之前的策略，玩家 $A$ 会把一个产生最大价值的数字单独放在一个盒子中，把剩余的数字全部放在另一个盒子中。

枚举$k=1\to n$，取最大值。

```c++
void solve()
{
    int n;
    std::cin >> n;
    std::vector<int> a(n);
    for(int i = 0; i < n; i++) std::cin >> a[i];
    std::sort(a.begin(),a.end());
    int ans = 0;
    for(int i = 0; i + 2 < n; i++)
        ans = std::max(ans, a[i + 1] - a[i] + a[n - 1] - a[i]);
    for(int i = n - 1; i - 2 >= 0; i--)
        ans = std::max(ans, a[i] - a[i - 1] + a[i] - a[0]);
    std::cout << ans << '\n';
}
```### D. Knowledge Cards （1:18）【1264/1750，-1】

#### 题意 (重复4)

给一个 $n\times m$ 的地图，在起点 $(1,1)$ 有 $k$ 个棋子形成一个栈，给定顺序。

将棋子从起点移动到终点 $(n,m)$，形成一个栈，从栈顶到栈底顺序为 $1\to k$

一颗棋子可以移动到相邻的格子。除了起点与终点一个格子最多只能有一颗棋子。棋子不能移入起点，不能移出终点。

询问能否通过任意步实现。

#### 分析 (重复4)

棋盘与华容道类似，只要有一个自由点，总能将棋子移动到任意一个位置。

于是题目转换为，对于 $t = k \to 1$，对于当前的 $t$ 在栈中出现的位置是前 $n\times m -3$，则把元素 $t$ 从栈中删除。

尝试用一个堆维护信息

```c++
void solve()
{
    int n,m,k;
    std::cin >> n >> m >> k;
    std::queue<int> q;
    std::priority_queue<int> Q;
    for(int i = 1; i <= k; i++)
    {
        int x;
        std::cin >> x;
        q.push(x);
    }
    while(k)
    {
        if(!Q.empty() && Q.top() == k)
        {
            Q.pop();
            k--;
        }
        else if(Q.size() == n * m - 3) break;
        else
        {
            Q.push(q.front());
            q.pop();
        }
    }
    if(k) std::cout << "TIDAK\n";
    else std::cout << "YA\n";
}
```### E. Hanging Hearts （2:05）【1050/2000，-3】

#### 题意 (重复5)

给一棵树，点权为 $a_i$（点权未知）

每次选择一个叶子节点 $v$，将点权 $a_v$ 加入一个队列 $q$
之后对于它的父亲节点 $u$ 进行操作 $a_u=\min(a_u,a_v)$
删除节点 $v$

当整棵树被删掉后，$score$ 为队列 $q$ 中最长不下降子序列的长度

点权是 $1\to n$ 的一个排列，求 $score$ 的最大值

#### 分析 (重复5)

考虑一条链的情况
只要将叶子节点的点权赋值为 $1$，最终的队列 $q$ 就只包含数字 $1$，$score=n$

考虑根节点 $root$ 有 $k$ 个子节点
不难发现，无论如何都无法让 $score=k+1$，因为最初我们会选择点权较小的叶子节点 $x$，刷新根节点的点权 $a_{root}'$ 将小于等于 $a_x$；接着选择另一个叶子节点 $y$，有 $a_y>a_x$，$\cdots$,最终选择根节点;
构成的序列为 $a_x,a_y,\cdots,a_{root}'$，其中 $a_{root}' \le a_x < a_y<\cdots$。$score = k$
即有多个子节点产生贡献的情况下，根节点无法产生贡献。

于是考虑对一棵子树 $u$，定义 $f[u]$ 表示这棵子树产生的最大价值是多少，$g[u]$ 表示 $u$ 节点不产生贡献的情况下，这棵子树产生的最大价值是多少。
有 $g[u] = \sum f[v]$,$f[u]=\max(g[u],length[u])$

```c++
std::vector<std::vector<int>> To;
std::vector<int> length,f,g;
void dfs(int u)
{
    length[u] = 1;
    int sum = 0;
    for(int v : To[u])
    {
        dfs(v);
        length[u] = std::max(length[u],length[v] + 1);
        g[u] += f[v];
    }
    f[u] = std::max(length[u],g[u]);
}
void solve()
{
    int n;
    std::cin >> n;
    To.resize(n + 1);
    length.resize(n + 1);
    f.resize(n + 1);
    g.resize(n + 1);
    for(int i = 2; i <= n; i++)
    {
        int u;
        std::cin >> u;
        To[u].push_back(i);
    }
    dfs(1);
    std::cout << f[1] << '\n';
}
